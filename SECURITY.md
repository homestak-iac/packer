# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by emailing:

**security@homestak.dev**

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 7 days
- **Resolution timeline**: Communicated after assessment

## Scope

This policy applies to the latest version of the project on the main branch.

## Disclosure

We follow coordinated disclosure. Please allow reasonable time for fixes before public disclosure.
